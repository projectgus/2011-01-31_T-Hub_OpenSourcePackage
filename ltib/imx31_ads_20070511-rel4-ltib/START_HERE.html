MX31ADS System Documentation Overview
 
Getting Started...

Instructions for connecting the i.MX31ADS board to a host system and building a project that can be used for program development.

Basic Instructions

  1. Install LTIB and the MX31ADS BSP
  2. Connect i.MX31ADS eth0 to network
  3. Connect UART0 to host computer COM port
  4. Configure and build LTIB project for the board
  5. Download and run built images on the target board
